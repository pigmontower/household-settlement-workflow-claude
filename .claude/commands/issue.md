**目的**: Issueを読み込んで壁打ち・議論を行う

---

## 引数

$ARGUMENTS: Issue番号（例: 8）

## 実行手順

1. `gh issue view $ARGUMENTS` でIssue内容を取得して表示
2. Issueの内容について壁打ち・議論を開始
3. 議論が一段落したら「この内容をIssueコメントとして投稿しますか？」とユーザーに確認
4. ユーザーが投稿を希望した場合:
   - 議論の要点をまとめたコメントを作成
   - CLAUDE.mdの署名ルールに従い、冒頭に署名を追加
   - `gh issue comment $ARGUMENTS --body "コメント内容"` で投稿

## 注意事項

- 壁打ち中は自由に議論し、ユーザーの質問に答えること
- コメント投稿は必ずユーザーの確認を得てから行うこと
- 投稿するコメントは議論の要点を簡潔にまとめたものにすること
