# 要件整理からの変更差分

> このドキュメントは、[01_requirements_gathering/ANALYSIS.md](../01_requirements_gathering/ANALYSIS.md) で整理した内容に対する、要件定義フェーズでの変更差分のみを記載しています。
> 詳細な検討経緯や採用案の詳細については、元の要件整理ドキュメントを参照してください。

---

## 業務フローの変更

### 明細登録フロー

| 項目 | 変更内容 |
|---|---|
| メニュー表示 | 「メニューを選択してください」メッセージを追加 |
| ボタン表記 | テキストからボタンであることを明示（`ボタン: [訂正][仕分け][登録]`） |
| [登録]ボタン | 新規追加。仕分けせずに直接スプレッドシートへ登録可能に |
| スプレッドシート操作 | シーケンス図にスプレッドシートへの書き込み処理を明記 |

### 清算確認フロー

| 項目 | 変更内容 |
|---|---|
| スプレッドシート操作 | シーケンス図に読み込み・書き込み処理を明記 |
| シート名フォーマット | `清算_yyyymmdd` / `清算済_yyyymmdd` を図中に明記 |
